Hobby project to learn to code assembly. It uses the RC2014 computer kit. It provides a simple shell like SCM (https://smallcomputercentral.com/small-computer-monitor/) but without the awesome debugging features. For the actual features list, please see changes.txt

Building is done with zasm (https://github.com/Megatokio/zasm).
